Semantics

Use l-flexbox when you wish want to use flexbox to layout items on the screen using floats or flexbox for browsers that can support flexbox.

Usage

  1. Create a "flex container" and add the l-flexbox class to this element.
  2. Create child-elements and add the l-flexbox--item class to these elements.
